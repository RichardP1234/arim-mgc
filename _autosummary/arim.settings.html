arim.settings

Settings use in this library.

Usage:

import settings as s

# Get parameter: print(s.SOME_PARAMETER)

# Change parameter: s.SOME_PARAMETER = ‘new_value’

# Alt access (the parameter is a string) print(s.__dict__[‘SOME_PARAMETER’])
